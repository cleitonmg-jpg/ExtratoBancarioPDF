ExtratoAI — Rodar em Rede Local (Windows)

1 Descobrir o IP da máquina servidora

Abra o Prompt de Comando ou PowerShell e execute:

ipconfig

Procure o adaptador ativo (Wi-Fi ou Ethernet) e anote o Endereço IPv4.

Exemplo: 192.168.1.10

2 Alterar o HOST no arquivo .env

Abra o arquivo .env na raiz do projeto e mude a linha HOST:

# de: HOST=127.0.0.1 # para: HOST=0.0.0.0
127.0.0.1 = só a própria máquina.
0.0.0.0 = aceita conexões de qualquer IP da rede.

3 Liberar a porta 5000 no Firewall Administrador

Abra o PowerShell como Administrador e execute:

New-NetFirewallRule -DisplayName "ExtratoAI - Porta 5000" `
  -Direction Inbound `
  -Protocol TCP `
  -LocalPort 5000 `
  -Action Allow

Para confirmar que a regra foi criada:

Get-NetFirewallRule -DisplayName "ExtratoAI - Porta 5000"
Se preferir o caminho gráfico: Painel de Controle → Windows Defender Firewall → Configurações Avançadas → Regras de Entrada → Nova Regra → Porta → TCP 5000 → Permitir.

4 Verificar se o PostgreSQL está rodando

No Prompt de Comando:

sc query postgresql-x64-17
O nome do serviço pode variar: postgresql-x64-16, postgresql etc.
Se o STATE não for RUNNING, inicie com:
net start postgresql-x64-17

5 Iniciar a aplicação

No Prompt de Comando dentro da pasta do projeto:

cd e:\EXTRATOBANCARIOPDF
npm run dev

Você verá a confirmação:

serving on http://0.0.0.0:5000

6 Acessar de outro computador na rede

Nos outros computadores, abra o navegador e acesse:

http://192.168.1.10:5000
Substitua 192.168.1.10 pelo IP anotado no Passo 1.

7 Resumo do .env para rede

PORT=5000 HOST=0.0.0.0 SESSION_SECRET=troque-por-string-segura # Banco de dados DB_TYPE=postgres DATABASE_URL=postgres://postgres:postgres@localhost:5432/extratoai

! Observações importantes

ItemDetalhe
PostgreSQL Só precisa ser acessível pela própria máquina (localhost). Os outros PCs acessam apenas o servidor Node.
Tipo de rede Se a rede for classificada como Pública no Windows, o Firewall pode bloquear mesmo com a regra. Mude para Privada em: Configurações → Rede → Propriedades.
IP fixo Recomendado fixar o IP no roteador (DHCP reserva) para o endereço não mudar ao reiniciar.
SESSION_SECRET Troque change-me por uma string aleatória longa antes de usar em rede.
Parar o servidor No terminal onde o servidor está rodando, pressione Ctrl + C.
